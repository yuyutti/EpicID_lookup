# EpicID(accountID)検索サイト

EpicID(accountID)をディスプレイネーム、FNに紐づいているPSNID,XboxIDから検索できます！

## 使い方

1. cloneなりDLなりしてローカル環境に持ってくる
2. ```npm i```して必要moduleをインストールする
3. .envファイルを作成し、fortniteapi.ioのAPIアクセスキーを入力する
4. ```node index.js```でサーバーを起動する

    3000ポートで起動します

fortniteapi.ioのAPIアクセスキーに関しては調べたらすぐ出てくるので各自行ってください
